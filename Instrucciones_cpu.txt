Para Uso de cpu:

Crear el ambiente conda:
    conda create --name doa_knn python=3.10

Activar entorno creado:
    conda activate doa_knn

Instalar las librerias necesarias:
    pip install -r requirements.txt

Verificar uso de version CPU:

    Debe estar comentada la linea: #from knn_fs_wrapper_nvidia import KnnFitness
    
    La linea activa debe ser: from knn_fs_wrapper_sklearn import KnnFitness

Ejecutar
    python run_experiment.py
